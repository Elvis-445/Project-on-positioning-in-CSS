Project on Positioning in CSS

No Positioning: This position in CSS Has no positioning meaning it does not move no matter the scrolling.

With Sticky Positioning: This means the element will scroll with the page until it reaches 20px from the top of the viewport, where it will then stay fixed

With Relative Positioning: Relative positioning in CSS allows you to adjust an element’s position relative to its normal position in the document flow.

With absolute Positioning: Absolute positioning in CSS allows you to place an element at a specific location within its containing element. When you set an element’s position property to absolute, it is removed from the normal document flow and positioned relative to its nearest positioned ancestor (an element with a position other than static).

With fixed Positioning: In CSS, fixed positioning is achieved by setting the position property to fixed. This positions the element relative to the viewport, meaning it stays in the same place even when the page is scrolled.
